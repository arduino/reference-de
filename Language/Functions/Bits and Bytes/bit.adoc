= bit()


// ÜBERSICHTSABSCHNITT STARTET
[#overview]
--

[float]
=== Beschreibung
Berechnet den Wert des angegebenen Bits (Bit 0 ist 1, Bit 1 ist 2, Bit 2 ist 4, etc.).
[%hardbreaks]


[float]
=== Syntax
`bit(n)`


[float]
=== Parameter
`n`: Das Bit, von dem der Wert berechnet werden soll.


[float]
=== Rückgabewert
Der Wert des Bits.
