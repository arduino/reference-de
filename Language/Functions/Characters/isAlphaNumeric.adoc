= isAlphaNumeric(thisChar)


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Beschreibung
Analysiert, ob ein Zeichen alphanumerisch ist. (Buchstabe oder Zahl). Gibt ``true`` zur√ºck, wenn das Zeichen eine Zahl oder ein Buchstabe ist.

[float]

----

----

[float]
=== Parameter
`thisChar`: Variable. *Erlaubte Datentypen:* char

[float]
=== Returns
`true`: Wenn das Zeichen alphanumerisch ist.
